# PROYECTO REPORTE SERVICIOS

Proyecto API de gestión de Reporte de servicios para taller mecánico desarrollado en python usando el framework fastapi, base de datos sqlite y reportlab para reportes pdf.

## COMANDOS DE EJECUCIÓN

- `uvicorn main:app --reload --port 8000`
- `python main.py`
- `python init_db.py`

### DIRECTRICES DEL CÓDIGO

- Utilizar el estándar PEP8 para la codificación.
- Documentar funciones y clases con docstrings estilo Google.
- Utilizar excepciones personalizadas para manejar errores específicos.
- Nombre de variables en inglés pero explicativas.
